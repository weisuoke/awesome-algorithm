# DP

动态规划(Dynamic Programming)。是求解决策过程(decision process)最优化的数学方法。

**思想**

感觉很像时高中数列的思想，给出首项，以及一个递推式子，让你求任意项的值。

步骤基本是： 寻找状态转移方程 => 建立合适的数据结构表 => 填表

**相关问题**

- 爬楼梯

- 打家劫舍

- 最大正方形

- 零钱兑换

- 不同路径

- 股票题状态机
